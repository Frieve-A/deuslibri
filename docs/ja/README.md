# DeusLibri ドキュメント

[English](../README.md)

DeusLibriドキュメントへようこそ。このディレクトリには、開発者とコンテンツ制作者向けの包括的なガイドが含まれています。

## 利用可能なドキュメント

### 日本語

- **[プロジェクト計画書](./PROJECT_PLAN.md)** - 完全な技術仕様、アーキテクチャ、実装ロードマップ
- **[執筆ガイド](./WRITING_GUIDE.md)** - 電子書籍の執筆・公開方法に関する包括的なガイド
- **[縦書きモード技術仕様書](./VERTICAL_MODE_SPEC.md)** - 日本語縦書き表示の実装仕様と動作原理

### English

- **[Project Plan](../PROJECT_PLAN.md)** - Complete technical specifications, architecture, and implementation roadmap
- **[Writing Guide](../WRITING_GUIDE.md)** - Comprehensive guide for book authors
- **[Vertical Mode Spec](../VERTICAL_MODE_SPEC.md)** - Japanese vertical writing mode technical specification

### その他の言語

他の言語への翻訳も歓迎します！詳細は[コントリビューションガイド](../../README.md#contributing)をご覧ください。

## クイックナビゲーション

### 開発者向け

DeusLibriの実装や、コードベースへの貢献を行う場合:
- まず[プロジェクト計画書](./PROJECT_PLAN.md)をお読みください
- 技術スタックとアーキテクチャを確認してください
- 実装ロードマップに従ってください

### 著者向け

DeusLibriで書籍を執筆する場合:
- まず[執筆ガイド](./WRITING_GUIDE.md)をお読みください
- ファイル構造とメタデータ形式を学んでください
- サンプル書籍を参考にしてください

## ドキュメントへの貢献

ドキュメントの改善は常に歓迎します！貢献するには:

1. タイポや小さな修正は、直接PRを送信してください
2. 大きな変更の場合は、まずissueを開いて議論してください
3. 新しい言語の翻訳を追加する場合は、既存のドキュメントと同じ構造を維持してください
4. 新しいドキュメントを追加した際は、このREADMEを更新してください
