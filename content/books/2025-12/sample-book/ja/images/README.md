# 画像フォルダ

このフォルダには、書籍の表紙や挿絵を配置します。

## 必要な画像

### 必須
- `cover.jpg` - 表紙画像（800x1200px以上推奨）

### オプション
- `illustration-01.jpg` - 本文中で参照される挿絵
- `illustration-02.jpg` - 本文中で参照される挿絵
- その他、必要に応じて画像を追加

## 画像の推奨仕様

- **形式**: JPEG（写真）またはPNG（イラスト・透過）
- **サイズ**: 幅1200px以上
- **ファイルサイズ**: 各500KB以下
- **命名規則**: 半角英数字とハイフン、小文字推奨

## 注意事項

このサンプル書籍では、実際の画像ファイルは含まれていません。
実際に執筆する際は、適切な画像を用意してください。

著作権に注意し、自分で作成した画像または使用許可のある画像のみを使用してください。
